Meine Infrastruktur

Übersicht meiner Systemlandschaft

Meine private Infrastruktur besteht aus einem leistungsfähigen Proxmox-Cluster mit folgenden Hauptkomponenten:

  • Proxmox VE Cluster (2 Nodes mit Ceph-Speicher)
  • Container (LXC):

    • Nextcloud

    • Trilium Notes
    • Code-Server (VS Code im Browser)
  • VMs:
    • Docker-Host für verschiedene Dienste

    • WireGuard VPN für sicheren Fernzugriff
